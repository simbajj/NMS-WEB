Windows 10 지원 종료에 따른 운영체제 업그레이드 안내
고객님은 현재 Windows 10 운영체제를 사용하고 있습니다.

2025년 10월 14일자로 Windows 10이 지원 종료됨에 따라 안전적이고 원활한 서비스 이용을 위해 Windows 11로 업그레이드 바랍니다.

지원 종료 후에도 피망 게임을 이용할 수 있으나 보안이 취약해질 수 있어 원활한 플레이가 어려울 수 있습니다.
안정적인 서비스를 위하여 아래 링크를 통해 Windows 11로 업그레이드하시기 바랍니다.
Windows 11 업그레이드하기
💡 Windows 11로 업그레이드 하는 법
1) 위 링크 클릭 후 "지금 다운로드" 버튼 클릭
이미지 추가 예정 - 협의 필요

2) 다운로드 파일 클릭
이미지 추가 예정 - 협의 필요

3) PC 상태 검사 앱 받기 클릭
이미지 추가 예정 - 협의 필요
❓ 혹시 도움이 더 필요하신가요?
업그레이드가 원활하지 않거나 설치가 되지 않을 경우 고객센터에서 도움드리겠습니다.
